## Idea:  
A **fancy** kanban board application like Trello

## Core Features:  
- Drag & Drop
- Creating, deleting & customizing columns   
- Adding team members to the tasks  
- Adding projects & Team members

## Additional Features:  
- Capacity of team members  
- Profile & login  
- Task priorisation
